# Sales Analysis Mini Project

This is a mini-project analyzing a small sales dataset using Python.

## Dataset
- `sales_data.csv`: Contains 10 rows of sales data including Date, Product, Units Sold, Unit Price, and Revenue.

## Analysis
- Python script: `analysis.py` or `analysis.ipynb`
- Visualizations using matplotlib:
  - Revenue over time
  - Units sold per product
  - Revenue per product

## Tools Used
- Python
- pandas
- matplotlib
